# Obligatorio 1: Sistemas Multiagente

En este proyecto exploramos cómo distintos algoritmos de aprendizaje logran (o no) adaptarse cuando se enfrentan en entornos competitivos y cooperativos. Implementamos desde cero agentes clásicos como Fictitious Play y Regret Matching, sumando también variantes de Reinforcement Learning (Independent Q-Learning y Joint-Action Learning). 

El corazón del trabajo fue poner a estos agentes a jugar entre sí en escenarios conocidos como Piedra-Papel-Tijera, Matching Pennies, Blotto y un ambiente de Foraging. A través de la experimentación, analizamos la evolución de sus estrategias y cómo llegan a equilibrarse (o desestabilizarse) dependiendo de qué algoritmo tenga el oponente enfrente.
